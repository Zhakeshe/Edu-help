✅ 2. Технологиялар бойынша талап

Сайтты төмендегі стекпен жаса:

Frontend: React + TailwindCSS

Backend: Node.js (Express)

Database: MongoDB (materials, classes, AI tools)

Auth: JWT (admin login)

File Upload: Multer (PDF/Word/PPT)
